# YYC³系统组件设计规范

## 1. 设计系统规范（design-system/）

### 1.1 颜色系统（color-system.md）
- 核心内容：
    - 品牌主色：#6252e9（基础值）、深阶#4a3dc7（深色模式）、亮阶#8b7ef7（交互强调）
    - 中性色：浅色模式（前景#111827、弱化#6b7280、边框#e5e7eb）、深色模式（前景#f9fafb、弱化#9ca3af、边框#374151）
    - 颜色模式规则：浅色 / 深色模式色值映射表、自定义模式可编辑范围（允许修改中性色，主色不可改）

### 1.2 排版规范（typography.md）
- 核心内容：
    - 字体体系：全局无衬线字体Inter（品牌指定），标题字重600、正文字重400
    - 字号层级：导航文字14px、弹窗标题18px、卡片标题16px、正文14px
    - 标语规范：首页左上角标语（中文 “万象归元于云枢｜深栈智启新纪元”、英文 “All Things Converge in Yunshu | Deep Stack Intelligence Usheres in a New Era”），字号16px，字重500

### 1.3 品牌标识规范（brand-identity.md）
- 核心内容：
    - Logo 使用场景：
        - 主 Logo：yyc3-logo.png（品牌核心展示区）
        - 网页标签：yyc3-pwa-icon.png（浏览器标签页）
        - 模式适配：浅色模式yyc3-logo-blue.png、深色模式yyc3-logo-white.png
        - 聊天框头像：“YYC³”（斜体，字号24px，颜色同主色）
    - Logo 切换规则：随页面颜色模式自动切换（检测prefers-color-scheme或用户设置）

### 1.4 组件样式规范（component-style.md）
- 核心内容：
    - 基础组件（按钮 / 图标）：
        - 按钮：圆角8px，主按钮内边距10px 20px，次要按钮内边距8px 16px
        - 图标：2px 线性圆角风格，导航栏图标20px、按钮内图标16px
    - 卡片组件：圆角12px，内边距16px，浅色阴影0 2px 6px rgba(0,0,0,0.04)、深色阴影0 2px 6px rgba(0,0,0,0.1)

### 1.5 布局规则（layout-rules.md）
- 核心内容：
    - 导航栏布局：左侧导航栏宽度240px（展开）/64px（仅图标），顶层导航栏高度64px
    - 弹窗定位：锚定触发源（如设置按钮），间距8px，边缘防溢出逻辑（自动反向显示）
    - 响应式适配：移动端左侧导航栏隐藏（点击汉堡按钮展开），弹窗宽度90%（最大360px）

## 2. 交互规范（interaction/）

### 2.1 导航栏交互（navigation-interaction.md）
- 核心内容：
    - 状态切换：选中按钮显示 “图标 + 文字”，未选中仅显示图标（切换动画200ms渐显）
    - 点击反馈：选中 / 未选中按钮点击时，背景色 100ms 闪显（浅色#f3f4f6/ 深色#374151）
    - 同步规则：左侧导航与顶层导航（代码工作台 / 智能编程 / 提示词库）选中状态颜色强制同步

### 2.2 弹窗交互（popover-interaction.md）
- 核心内容：
    - 触发与关闭：设置按钮点击展开，支持 “关闭图标 / 外部点击 / ESC 键” 关闭（消失动画300ms缩放 + 淡出）
    - 内部元素交互：
        - 复选框：点击切换选中状态（勾选动画150ms）
        - 下拉选项：hover 背景变色（200ms渐变），选中时高亮（颜色同主色）

### 2.3 全局交互统一（global-interaction.md）
- 核心内容：
    - 点击反馈：所有按钮 / 图标点击时，Y 轴下沉1px（100ms），背景色加深 10%
    - hover 效果：统一200ms颜色渐变 + 阴影微扩（主按钮阴影扩散2px）
    - 颜色模式切换：无延迟同步所有组件色值（优先读取用户设置，其次系统默认）

## 3. 组件文档（components/）

### 3.1 导航组件（navigation-components.md）
- 左侧导航栏：
    - 功能：包含设置按钮等入口，支持展开 / 折叠
    - 样式：选中 / 未选中状态色值（见 1.1 颜色系统）
    - 特殊元素：设置按钮（图标⚙️，点击展开列表及高级设置弹窗）
- 顶层导航栏：
    - 功能：包含代码工作台 / 智能编程 / 提示词库入口
    - 样式：与左侧导航栏状态样式同步

### 3.2 弹窗组件（popover-components.md）
- 设置弹窗（含高级设置）：
    - 结构：标题区（可选）+ 选项列表 + 高级设置表单 + 操作按钮区
    - 样式：容器圆角12px，内边距16px（见 1.4 组件样式）
    - 表单元素：复选框边框色（浅色#d1d5db/ 深色#596474）、下拉选项选中背景（见 1.1 颜色系统）

### 3.3 浮动按钮弹窗（floating-button.md）
- 功能：首页全局操作入口（见图示 8）
- 样式：圆形按钮（直径56px），背景#6252e9，图标24px #ffffff
- 交互：点击展开功能列表（动画300ms），支持拖拽（吸附底层栅格）

## 4. 开发指南（dev-guide/）

### 4.1 组件集成（integration-guide.md）
- 导航栏集成：左侧 / 顶层导航栏组件注册示例（含状态同步逻辑）
- 弹窗集成：设置弹窗与左侧导航按钮的关联代码（锚定定位实现）
- 颜色模式适配：通过 CSS 变量实现模式切换（--primary-color动态赋值）

### 4.2 API 参考（api-reference.md）
- 导航状态 API：setNavActive(id)（设置选中状态）、syncNavColors()（同步导航颜色）
- 弹窗控制 API：openPopover(target, content)（打开弹窗）、closePopover()（关闭弹窗）
- 颜色模式 API：getColorMode()（获取当前模式）、setCustomColor(var, value)（自定义颜色）

## 说明
