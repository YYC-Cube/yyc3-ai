# 快速开始指南

## 5分钟上手

### 步骤1: 了解界面 (1分钟)

打开应用后,您会看到顶部有8个主要标签页:
- **AI 对话**: 与AI助手对话
- **代码工作台**: 编写和预览代码
- **智能编程**: AI辅助编程
- **项目管理**: 管理项目
- **代码审查**: 代码质量分析
- **性能监控**: 性能监控
- **学习路径**: 学习计划
- **提示词库**: AI提示词管理

### 步骤2: 编写第一段代码 (2分钟)

1. 点击"智能编程"标签页
2. 在编辑器中输入以下代码:

\`\`\`javascript
function hello() {
  return "Hello, AI!"
}

console.log(hello())
\`\`\`

3. 右侧预览窗口会自动显示结果
4. 查看AI提供的代码建议

### 步骤3: 体验AI功能 (2分钟)

1. 点击"AI 对话"标签页
2. 输入: "帮我写一个计数器组件"
3. AI会生成完整的代码
4. 复制代码到"代码工作台"查看效果

## 核心功能演示

### 功能1: 实时代码预览

\`\`\`javascript
// 在代码工作台中输入
function App() {
  return <div>Hello World</div>
}
\`\`\`

右侧预览窗口会立即显示结果。

### 功能2: AI代码建议

在智能编程环境中,AI会自动:
- 检测代码错误
- 提供改进建议
- 识别学习机会
- 评估代码质量

### 功能3: 代码审查

1. 编写代码
2. 切换到"代码审查"
3. 查看详细分析报告
4. 一键应用修复
